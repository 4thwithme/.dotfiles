# Key Bindings for VSCode

## Bars Key Bindings

- ⌘+b: Toggle Side Bar Visibility
- ⌘+⌥+b: Toggle Secondary Side Bar
- ⌘+j: Toggle Terminal

## Editor Key Bindings

- ⌘+⇧+p: Show Command Palette
- ⌘+⇧+e: Show Explorer
- ⌘+⇧+f: Show Search
- ⌘+⇧+h: Show Search With Replace
- ⌘+⇧+d: Show Debug
- ⌘+⇧+x: Show Extensions
- ⌘+⇧+m: Show Problems
- ⌘+⇧+u: Show Output
- ⌘+⇧+y: Show Debug Console
- ⌘+⇧+v: Show Markdown Preview

## Copilot

- ⌘+i: Inline Copilot editor
- ⌘+⇧+i: Chat Copilot editor

## Vim Key Bindings

### Modes

- n: Normal Mode
- i: Insert Mode
- v: Visual Mode
- V: Visual Line Mode
- Ctrl+v: Visual Block Mode
- o: Open Line Below
- O: Open Line Above
- a: Append Mode
- A: Append to End of Line
- I: Insert at Beginning of Line
- R: Replace Mode
- s, S: Select Mode
