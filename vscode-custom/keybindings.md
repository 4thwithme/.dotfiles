# Key Bindings for VSCode

## Bars Key Bindings

- ⌘+b: Toggle Side Bar Visibility
- ⌘+⌥+b: Toggle Secondary Side Bar
- ⌘+j: Toggle Terminal
- ⌘+.: Open Settings

## Editor Key Bindings

- ⌘+⇧+p: Show Command Palette
- ⌘+⇧+e: Show Explorer
- ⌘+⇧+f: Show Search
- ⌘+⇧+h: Show Search With Replace
- ⌘+⇧+d: Show Debug
- ⌘+⇧+x: Show Extensions
- ⌘+⇧+m: Show Problems
- ⌘+⇧+u: Show Output
- ⌘+⇧+y: Show Debug Console
- ⌘+⇧+v: Show Markdown Preview
- ⌘+g: Show Git

## Copilot

- ⌘+i: Inline Copilot editor
- ⌘+⇧+i: Chat Copilot editor

## Vim Key Bindings

### Modes

- n: Normal Mode
- i: Insert Mode
- v: Visual Mode
- V: Visual Line Mode
- Ctrl+v: Visual Block Mode
- o: Open Line Below
- O: Open Line Above
- a: Append Mode
- A: Append to End of Line
- I: Insert at Beginning of Line
- R: Replace Mode
- s, S: Select Mode

## Vim Motions

- h, left: Move Left
- j, down: Move Down
- k, up: Move Up
- l, right: Move Right
- w: Move to Next Word
- b: Move to Previous Word
- e: Move to End of Word
- 0: Move to Beginning of Line
- $: Move to End of Line
- gg: Move to Beginning of File
- G: Move to End of File
- D: jump down half a page
- U: jump up half a page
- f + r: Move to Next Occurrence of Character `r`
- F + r: Move to Previous Occurrence of Character `r`
