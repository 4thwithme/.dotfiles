B2B SaaS platform for tracking engineer performance within companies.

Data sources: Jira (tickets/velocity) and Git (commits, PRs, code activity).

**Why:** Helps engineering managers and companies get data-driven insight into individual and team performance.

**How to apply:** New features will likely involve integrating with Jira and Git APIs, aggregating stats, and surfacing metrics per engineer/team. Keep this context in mind when designing data models and modules.
